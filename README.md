# 💳 Credit Card Fraud Detection (ML + Streamlit App)

## 📌 Overview
This project detects fraudulent credit card transactions using a machine learning model. It analyzes transaction data and classifies whether a transaction is **fraudulent or legitimate**. A simple **Streamlit web app** is included for real-time predictions.

---

## 🚀 Features
- Detects fraudulent transactions using ML
- Data preprocessing and feature scaling
- Trained classification model
- Interactive Streamlit web app
- Real-time prediction interface

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn (optional for EDA)
- Streamlit
